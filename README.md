## 🚀 Key Features

### 🧠 Neural Intelligence Hub
Menggunakan pemprosesan visi komputer untuk mengenal pasti objek, wajah, dan aktiviti secara autonomi merangkumi 4 sektor utama:
* **Industry:** Integrasi IoT & LPR (License Plate Recognition).
* **Retail:** Analitik demografi & People Counting.
* **Corporate:** Pengurusan akses VIP & Face Recognition Terminals.
* **Home:** Keselamatan perimeter dengan integrasi bot Telegram.

### 🌐 Global Node Mapping
Visualisasi geospatial yang memaparkan koordinat peranti (Nodes) di seluruh dunia (atau kawasan tempatan) dengan sambungan penyulitan AES-256 yang selamat.

### ⚡ Real-Time Diagnostics
Monitor kesihatan sistem yang memaparkan:
* **AI Compute Load:** Beban pemprosesan CPU/GPU semasa.
* **Global Sync Speed:** Latensi rangkaian antara nod.
* **Thermal Monitoring:** Status suhu perkakasan (hardware).

### 🖥️ Secure Shell Terminal
Sistem log automatik yang merakam setiap aktiviti rangkaian, cubaan pencerobohan, dan status penyegerakan pangkalan data secara masa nyata.

## 🔒 Security Protocol

Projek ini menekankan privasi data melalui:

  * **End-to-End Encryption:** Semua data dari nod ke pelayan disulitkan.
  * **Manual Override:** Keupayaan untuk memutuskan sambungan nod secara manual melalui Control Panel.
  * **Intrusion Alert:** Sistem amaran automatik (Part 9) jika berlaku cubaan akses tidak sah.
